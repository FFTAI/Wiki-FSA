\doxysubsection{Detailed Description}
Maintains a registry of network-\/connected modules and returns \mbox{\hyperlink{class_fourier_1_1_group}{Group}} objects to the user. 

Only one \mbox{\hyperlink{class_fourier_1_1_lookup}{Lookup}} object is needed per application.
